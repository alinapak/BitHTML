HTML

Teorija

HTML (Hyper Text Markup Language) yra žymėjimo kalba, kuri naudoja specifinę sintaksę arba žymėjimą, kad apibūdintų tinklalapio struktūrą naršyklėje. HTML elementai paprastai turi atidarymo ( < ) ir uždarymo ( / > ) žymes, kurios supa ir įprasmina turinį (angl. content). Skirtingi elementai gali apibūdinti tekstą kaip antraštę (heading (h1-h6)), pastraipą (p) ar sąrašo elementą (li).

Praktika

Bandyti įgyvendinti HTML patiems, nei apie šią žymėjimo kalbą skaityti, yra kur kas įdomiau. Šioje skiltyje pateiksiu praktinį HTML, t.y. pareiksiu pavyzdį, kaip įkelti paveikslėlį į, tarkime, kuriamą puslapį.
