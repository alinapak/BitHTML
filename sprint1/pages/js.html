JavaScript

HTML ir CSS kontroliuoja kuriamo puslapio turinį ir stilių. JavaScript programavimo kalba yra skirta padaryti tą turinį interaktyviu (sąveikaujančiu). JavaScript turi kintamuosius (variables), arrays, objects, loops ir funkcijas (functions).

JavaScript yra "scenarijų" kalba, kuri kaip minėta naudojama norint padaryti tinklalapius interaktyviais. Tai viena iš pagrindinių interneto technologijų, einančių kartu su HTML ir CSS ir yra palaikoma visų šiuolaikinių naršyklių.
